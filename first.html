Arrays

Arrays are a fundamental linear data structure that stores elements of the same type in contiguous memory locations.
 Accessed by index, arrays offer fast retrieval but have a fixed size.

Linked List

Linked Listsare dynamic data structures where elements, or nodes, contain data and a reference to the next node.
allowing for flexible size and efficient insertions and deletions.

Stack

Stackfollow the Last In, First Out (LIFO) principle restricting operations to adding and removing elements from the same end (top). They are often used for managing function calls and undo mechanisms.
